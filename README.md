## Prerequisites

- **[Go](https://go.dev)**: **1.18+**.

## Installation

Simply append the following import to your code, and then `go [build|run|test]`
will automatically fetch the necessary dependencies:

```go
import "github.com/HarutakaMatsumoto/mathematics_go"
```
